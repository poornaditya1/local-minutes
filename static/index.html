macOS system audio setup

1. Install a virtual audio device such as BlackHole 2ch, Loopback, or VB-Cable.
2. Open Audio MIDI Setup and create a Multi-Output Device that includes your headphones or speakers plus the virtual device.
3. Set that Multi-Output Device as your Mac output, then choose the virtual device as System audio in this app.
4. Grant microphone permission to Terminal, Python, or your IDE when macOS prompts.
